\# Chorizoli - Sistema de Gestión de Embutidos



\*\*Materia:\*\* Desarrollo WEB Backend  

\*\*Estudiante:\*\* JHOEL YESID MAMANI CONDORI  



\## Descripción

Sistema para la gestión de productos embutidos artesanales, con autenticación, reportes PDF, gráficos estadísticos y logs de acceso.



\## Tecnologías

\- Frontend: React + TypeScript + Vite + Tailwind CSS

\- Backend: NestJS + PostgreSQL + TypeORM

\- Autenticación: JWT + bcrypt

\- Reportes: PDFMake

\- Gráficos: Chart.js
